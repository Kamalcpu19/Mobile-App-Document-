# Vehicle Search & Tracking

### Features

- **Vehicle Search** — find vehicles by registration number or customer
- **Service Stage Tracking** — monitor current workflow stage
- **Vehicle Progress Monitoring** — view real-time status updates
- **Service History** — access previous service records
- **Estimate Actions** — create or update estimates

### Expected Result

- Vehicles should display in correct stages
- Vehicle progress should update dynamically
