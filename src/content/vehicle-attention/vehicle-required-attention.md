# Vehicle Required Attention Module

### Description



- Tracks all vehicles currently under service and their workflow stages.

- Features

- Vehicle Search

- Service Stage Tracking

- Vehicle Progress Monitoring

- Service History

- Estimate Actions

- Create RO Access

- Profile Access

### Status Categories



- Estimation Request

- Estimate

- Spares Pending

- Approval Pending

- Work In Progress

- Ready For Delivery

- Invoice

- Delivered

### Flow



1. Vehicle Required Attention
2. View In-Service Vehicles
3. Filter by Service Stage
4. Track Vehicle Progress
5. Service History / Estimate Actions

### Expected Result



- Vehicles should display in correct stages.

- Vehicle progress should update dynamically.
