# Customer Messages Module

### Description



- The Customer Messages module allows users to manage customer communications and service-related conversations.

- Features

- Customer Conversations

- AI Complaint Detection

- Quick Reply

- Call Option

- Camera Attachment

- Create RO Access

- Profile Access

### Communication Flow



1. Customer Messages
2. View Customer Conversations
3. AI Complaint Detection
4. Quick Reply / Call / Camera Actions

AI Complaint Detection

### Description



- The application uses AI to detect customer complaints automatically from customer messages.

- Detected complaints are attached to the Job Card automatically.

### Expected Result



- AI should detect complaints correctly.

- Complaints should attach to Job Cards successfully.
