# Profile Access Module

## Feature: User Profile & Workshop Settings

### Description



- The Profile module allows users to view workshop information, subscription details, enabled capabilities, privacy settings, application settings, and account-related configurations.

- Users can access the Profile module from the top-right Profile Icon available across all major modules in the application.

### Available Sections



- User Profile Information

- Workshop Information

- Contact & Location Details

- Plan & Subscription

- Enabled Capabilities

- Package Features

- Privacy & Data Settings

- App Language Settings

- Workspace Settings

- App Version Information

- Sign Out Option
