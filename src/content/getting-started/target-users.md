# Target Users

The mobile application is built for the following user roles:

- **Garage Owners** — oversee workshop operations and business performance
- **Service Advisors** — manage customers, appointments, and job cards
- **Technicians** — perform inspections, service work, and updates
- **Billing Staff** — handle invoices, payments, and financial follow-ups
