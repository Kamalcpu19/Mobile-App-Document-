# Pre-Inspection Completion

### Workflow



1. User completes inspection activities.

2. User clicks on Pre-Inspection Done.

3. The Repair Order was created successfully.

4. Create RO process closes automatically.

5. Vehicle moves to the Vehicle Required Attention section.

### Expected Result



- RO should be created successfully

- The vehicle should appear in the Vehicle Required Attention tracking
