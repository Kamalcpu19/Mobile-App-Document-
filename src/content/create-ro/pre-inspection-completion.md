# Pre-Inspection Completion

### Flow



- User completes inspection activities.

- User clicks on Pre-Inspection Done.

- Repair Order is created successfully.

- Create RO process closes automatically.

- Vehicle moves to Vehicle Required Attention section.

### Expected Result



- RO should create successfully.

- Vehicle should appear in Vehicle Required Attention tracking.
