# Create Account

### Create Account Feature

The application provides a Create Account option for new users.

### Create Account Flow

1. User clicks on Create Account
2. User enters required registration details
3. User submits the registration form
4. System validates entered details
5. Account is created successfully
6. User can log in using newly created credentials

### Expected Result

New users should be able to create an account successfully.
