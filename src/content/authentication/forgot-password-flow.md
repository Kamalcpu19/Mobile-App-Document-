# Forgot Password Flow

### Forgot Password Flow



1. User clicks on Forgot Password.

2. System redirects user to password recovery screen.

3. User enters registered mobile number or email address.

4. OTP verification process is completed.

5. User creates new password.

6. User logs in using updated password.
