# User Sign In

## Feature: User Sign In

### Description

The Login page allows registered users to securely access the mobile application using their username and password credentials.

### Available Options

- Username Input Field
- Password Input Field
- Sign In Button
- Forgot Password Option
- Create Account Option

### Login Flow

1. User opens the mobile application
2. User enters registered username
3. User enters password
4. User clicks on the Sign In button
5. System validates the credentials
6. On successful authentication, the user is redirected to the Dashboard/Home screen
