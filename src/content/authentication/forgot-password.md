# Forgot Password

### Forgot Password Flow

1. User clicks on Forgot Password
2. System redirects the user to password recovery screen
3. User enters registered mobile number/email
4. Verification process is completed through OTP or registered verification method
5. User creates a new password
6. User logs in using the updated password

### Expected Result

Users should be able to reset forgotten passwords successfully.
