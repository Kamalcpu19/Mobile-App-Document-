# Landing Page / Dashboard

## Feature: Landing Page Dashboard

### Description



- The Landing Page acts as the main dashboard of the mobile application. After successful login, users are redirected to this screen where they can view important reminders, pending activities, and quick actions related to workshop operations.

### Available Sections



- Customer Messages Reminder

- Appointment Reminder

- Vehicle Attention Reminder

- Pending Payments Reminder

- Create RO (Repair Order) Quick Action

- Profile Icon Access

### Dashboard Information Cards

1. Customer Messages

### Description



- Displays the number of customer messages awaiting response.

### Expected Result



- Users should be able to identify pending customer communications quickly.

2. Appointments Scheduled

### Description



- Displays the total number of appointments scheduled for the current day.

### Expected Result



- Users should be able to track daily appointment schedules.

3. Vehicles Requiring Attention

### Description



- Displays the count of vehicles that require service attention or follow-up.

### Expected Result



- Users should be able to monitor pending vehicle activities.

4. Customer Payments Pending

### Description



- Displays the total pending customer payments.

### Expected Result



- Users should be able to identify pending payment collections.
