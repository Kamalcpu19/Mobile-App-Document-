# Landing Page Dashboard

## Feature: Landing Page Dashboard

### Description

The Landing Page acts as the main dashboard of the mobile application. After successful login, users are redirected to this screen where they can view important reminders, pending activities, and quick actions related to workshop operations.

### Available Sections

- Customer Messages Reminder
- Appointment Reminder
- Vehicle Attention Reminder
- Pending Payments Reminder
- Create RO (Repair Order) Quick Action
- Profile Icon Access

### Dashboard Information Cards

The dashboard displays information cards for each operational area. Select a topic from the sidebar to view detailed documentation for each card.
