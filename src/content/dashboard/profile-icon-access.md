# Profile Icon Access

### Description

The dashboard provides a **Profile Icon** in the top-right corner, available across all major modules in the application. Users can tap this icon to access workshop information, subscription details, settings, and account options.

### Expected Result

Users should be able to navigate to the Profile module successfully from the dashboard.

For full profile documentation, see **Profile Access** module in the sidebar.
