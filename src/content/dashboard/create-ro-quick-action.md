# Create RO Quick Action

### Description

A quick action button is provided on the dashboard (and across all modules) to create a new Repair Order (RO).

The Create RO module allows users to:

- Capture vehicle details
- Perform inspection activities
- Add customer complaints
- Create Repair Orders
- Track service workflow

### Create RO Flow

1. Vehicle Number Detection
2. Vehicle Image Capture
3. Pre-Inspection Checklist
4. RO Creation Complete

### Related Module

For step-by-step Create RO documentation, see **Create RO (Repair Order)** module in the sidebar.
