# Profile Access

### Description



- Users can access the Profile module from every major module in the application through the Profile Icon available in the top-right corner.

**The Profile module provides quick access to:**

- User Profile Information

- Workshop Information

- Subscription Details

- Enabled Features

- Privacy Settings

- Language Settings

- Workspace Settings

- Sign Out Option

### Expected Result



- Users should navigate to the Profile module successfully from any module.
